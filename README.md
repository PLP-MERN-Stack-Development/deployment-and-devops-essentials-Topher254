## Local Development
1. Backend: `cd server && npm run dev`
2. Frontend: `cd client && npm run dev`

## Deployment
- Backend: Deployed on Render
- Frontend: Deployed on Vercel
- Database: MongoDB Atlas

## Features
-  React frontend with Vite
-  Express.js backend
-  MongoDB database
-  CORS enabled
-  Production deployment
-  Basic health checks
